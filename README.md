# Лабораторна робота №1: Web-дизайн
**Розробка статичного інтерфейсу Web-додатка**

## Інформація про студента
- **ПІБ:** Іваха Богдан Миколайович
- **Група:** КВ-32

## Завдання
1. Ознайомитись із засобами мов HTML5 та CSS3.
2. Навчитись застосовувати бібліотеки (використано Tailwind CSS) щодо швидкої побудови Web-інтерфейсу користувача.
3. Розробити статичні Web-сторінки додатку та перевірити коректне їх відображення на екранах різного розміру.
4. Тематика додатка: **Блог з коментарями**.

## Склад проєкту
- `index.html` — сторінка "Про додаток" (з емблемою, описом та інформацією про розробника)
- `app.html` — робоча сторінка додатка (блог про подорожі та каву з коментарями)
- `profile.html` — сторінка профілю користувача зі статистикою та особистими даними
- `login.html` — форма входу в систему
- `register.html` — форма реєстрації нового користувача
- `style.css` — файл користувацьких стилів для глобальних налаштувань (::selection, scroll-behavior)
- `readme.md` — опис проєкту та технічна інформація
