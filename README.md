## Склад проєкту
- `index.html` — сторінка "Про додаток" (з емблемою та описом)
- `app.html` — робоча сторінка додатка (блог та коментарі)
- `profile.html` — сторінка профілю користувача
- `login.html` — форма входу
- `register.html` — форма реєстрації
- `style.css` — файл користувацьких стилів (відповідно до вимог)
- `readme.md` — опис проєкту
