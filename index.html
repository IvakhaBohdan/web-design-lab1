Вітаємо у "МійБлог"!

Це сучасна платформа для обміну думками та ідеями. Наш Web-додаток дозволяє користувачам створювати власні дописи, читати статті інших авторів та активно обговорювати їх у коментарях. Проєкт створено в рамках лабораторної роботи з використанням HTML5, CSS3 та фреймворку Tailwind CSS для забезпечення ідеального відображення на будь-яких пристроях.
